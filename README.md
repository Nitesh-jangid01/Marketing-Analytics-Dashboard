# 📊 Marketing Analytics Dashboard (Power BI)
Power BI dashboard analyzing marketing campaigns, channel performance, and influencer ROI.

## 🔍 Overview

This project presents an end-to-end marketing analytics solution built using Power BI. It analyzes campaign performance, channel effectiveness, and influencer marketing ROI.

---

## 🚀 Key Features

* Campaign performance tracking
* Channel-wise conversion analysis
* Influencer ROI evaluation
* Executive summary with business insights
* Interactive filters and drill-through

---

## 🧩 Dataset

The dataset includes:

* Clients
* Campaigns
* Channels
* Campaign Performance
* Influencers
* Influencer Campaigns

---

## 📊 Dashboards

### 1️⃣ Campaign Performance Dashboard

* Total Spend, Conversions, CTR, ROI
* Channel performance
* Monthly trends
* Client performance table

### 2️⃣ Influencer Marketing Dashboard

* Influencer ROI & cost analysis
* Top influencers
* Cost vs conversion scatter plot

### 3️⃣ Executive Summary Dashboard

* High-level KPIs
* Top campaigns
* Business insights

---

## 🧠 Key Insights

* Instagram drives highest conversions with lower cost
* Campaign_7 shows highest ROI
* Influencer campaigns contribute significantly to total conversions

---

## 🛠 Tools Used

* Power BI
* SQL (Data Modeling)
* DAX (Measures & KPIs)
